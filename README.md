# NTV Live PiP Fix

Safari workaround scripts for keeping Picture-in-Picture active on the NTV live stream page.

- Live page: [news.ntv.co.jp/live](https://news.ntv.co.jp/live)
- NTV News: [news.ntv.co.jp](https://news.ntv.co.jp/)
- Detailed usage: `scripts/README.md`

## Contents

- `scripts/ntv-live-pip-fix.user.js` - userscript workaround.
- `scripts/ntv-live-pip-fix.scpt` - compiled AppleScript workaround for Safari.
- `scripts/run-ntv-live-pip-fix.command` - double-click launcher for the AppleScript variant.
