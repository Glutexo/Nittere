# Project Rules

- After every code or file change, create a git commit.
- Do not reuse or overwrite the user's currently open windows; open a new window or tab instead.
- Do not troubleshoot by taking screenshots.
- Only use Safari as the browser; do not use other browsers.
- Safari can be automated with AppleScript when browser scripting is useful.
- The `gh` command is available for GitHub-related work.
